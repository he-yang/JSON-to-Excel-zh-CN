1. 简介

English

WTSolutions 推出的 JSON 转 Excel 工具集，可将扁平结构和嵌套结构的 JSON 数据转换为 Excel 文件。该工具集提供了 “JSON 转 Excel” 的全场景解决方案，包括 Excel 加载项、Web 应用程序：

  • Web 应用：直接在浏览器中进行 JSON 转 Excel 操作。

  • Excel 加载项：在 Excel 环境中实现 JSON 转 Excel，与 Excel 无缝集成。

  • WPS 加载项：在 WPS 环境中实现 JSON 转 WPS 工作表，与 WPS 无缝协作。

  • API：通过HTTPS POST请求在API中转换JSON到Excel

  • MCP服务器：在MCP服务器中转换JSON到Excel
