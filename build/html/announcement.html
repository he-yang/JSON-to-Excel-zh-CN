公告：JSON-to-Excel 功能调整通知

尊敬的 JSON-to-Excel 用户：

您好！为了更好地满足用户需求，提供更优质的服务，我们决定对 JSON-to-Excel 的免费版和付费版功能范围进行调整。以下是具体变化：

功能调整内容

免费版调整

  • 功能扩展：免费版现在可以使用所有之前仅限付费版的功能，包括但不限于：

    • 批量处理：从本地计算机加载多个 JSON 文件

    • 批量处理：从 Web URL 加载 JSON 文件

    • 多种嵌套 JSON 分隔符（下划线、双下划线、正斜杠）

    • 自定义嵌套对象的最大深度（1-20）

  • 使用限制：免费版单次最多转换 6 行数据

付费版调整

  • 功能保持：付费版仍然可以使用所有高级功能

  • 使用限制：付费版无行数限制，可以转换任意数量的数据

  • 其他权益：付费版用户仍然享受无广告体验

生效时间

本次调整将于 2026 年 3 月 14 日正式生效。

对用户的影响

  • 免费版用户：您将能够使用更多高级功能，但单次转换的数据行数将限制为 6 行

  • 付费版用户：您的使用体验将保持不变，仍然可以无限制地使用所有功能

如何升级

如果您需要无限制地使用所有功能，建议您订阅我们的付费版。付费版订阅价格如下：

  • 月度：19.9 元/月（不含税）

  • 季度：26.0 元/季度（不含税）

  • 半年：36.0 元/半年（不含税）

  • 年度：44.0 元/年（不含税）

其他货币价格：

  • USD: $2.66/月, $3.66/季度, $4.99/半年, $5.99/年

  • HKD: HK$21.8/月, HK$28.0/季度, HK$39.0/半年, HK$48.0/年

联系方式

如果您对本次调整有任何疑问或建议，欢迎通过以下方式联系我们：

  • 电子邮件：he.yang@wtsolutions.cn

我们将继续努力改进 JSON-to-Excel，为您提供更好的服务体验！
